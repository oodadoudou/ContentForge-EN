# ContentForge

**[English Version / 英文版本](https://github.com/oodadoudou/ContentForge-EN) | [中文版本 / Chinese Version](https://github.com/oodadoudou/ContentForge)**

一个全面的数字内容处理工具包，专为漫画、电子书和文件管理而设计。

## ✨ 核心功能

- **📚 内容获取**: 自动从特定网站下载已购买的漫画资源
- **🎨 漫画处理**: 将零散漫画图片转换为优化的PDF文件，支持智能合并和分割
- **📖 电子书工坊**: 创建、转换和编辑EPUB/TXT文件，支持样式美化和格式化
- **🛠️ 文件修复**: 修复常见问题如编码错误和封面显示问题
- **🗂️ 文件库管理**: 通过AI翻译和加密功能自动整理文件
- **🌐 通用下载器**: 从支持的网站下载内容
- **⚙️ 系统化管理**: 统一的设置和依赖管理

## 🚀 快速上手

### 1. 安装依赖软件包

**自动安装（推荐）：**
1. 打开终端并导航到项目目录
2. 运行：`python main.py`
3. 选择 `6. 系统设定与依赖`
4. 选择 `3. 安装/更新项目依赖`

**手动安装：**
```bash
pip install -r requirements.txt
```

### 2. 设定预设工作目录

首次运行时，程序会引导您设定一个默认工作目录，用于所有操作。

### 3. 配置AI翻译（可选）

对于文件库管理功能：
1. 运行 `python main.py`
2. 进入 `6. 系统设定与依赖`
3. 选择 `2. 配置 AI 翻译 API`

## 📖 使用方法

1. 打开终端
2. 导航到项目目录：`cd /path/to/ContentForge`
3. 运行：`python main.py`
4. 根据菜单选择所需功能

## 🔧 模块详解

### 内容获取
- 从支持的平台下载已购买的漫画
- 自动登录凭证管理

### 漫画处理
- 将图片文件夹转换为优化的PDF
- 智能合并、分割和质量优化
- 无缝阅读体验

### 电子书工坊
- **创建**: TXT/Markdown → EPUB/HTML（带样式）
- **转换**: EPUB ↔ TXT
- **编辑**: 内容替换、样式美化、繁简转换
- **管理**: 合并/分割EPUB文件

### 文件修复
- 修复EPUB封面显示问题
- 解决TXT编码问题
- 纠正竖排版式格式

### 文件库管理
- 自动整理散乱文件
- AI驱动的文件夹翻译和重命名
- 文件/文件夹加密打包

### 通用下载器
- 从支持的网站下载小说
- 自动章节收集和打包
